# Vida Mais Itapira - Sistema de Formulários Mobile

## Sobre o Projeto
Este projeto foi desenvolvido em parceria com a **FATEC** e o programa **Vida Mais Itapira**, com o objetivo de modernizar a coleta de respostas dos associados.  

Atualmente, os formulários são aplicados **em papel**, o que dificulta a organização, o armazenamento e a análise dos dados. Este sistema propõe uma **solução mobile**, permitindo que os associados respondam os formulários diretamente pelo aplicativo, de forma prática e eficiente.

## Funcionalidades Principais
- Responder formulários de forma **digital e intuitiva**.
- Substituição do processo manual em papel, evitando perdas de informações.
- Armazenamento seguro das respostas no **Firebase**.
- Visualização de resultados, exportação de dados e gráficos para análise dos formulários preenchidos.

## Tecnologias Utilizadas
- **React Native** – para desenvolvimento do aplicativo mobile.
- **Firebase Firestore** – banco de dados em tempo real.
- **Expo** – para facilitar o desenvolvimento e deploy do aplicativo.
- **React Native Chart Kit** – geração de gráficos de forma simples e interativa.

## Objetivo
O sistema busca **agilizar e digitalizar o processo de coleta de dados**, garantindo maior praticidade para os associados e eficiência para a gestão do programa Vida Mais Itapira.
